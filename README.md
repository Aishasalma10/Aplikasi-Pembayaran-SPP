# Aplikasi Pembayaran SPP Berbasis Website
Aplikasi Pembayaran SPP merupakan sistem informasi berbasis website yang dirancang untuk membantu administrasi sekolah dalam mencatat, mengelola, dan menyimpan data pembayaran SPP secara terstruktur dan digital.

## Project Background
Aplikasi ini bertujuan untuk menggantikan sistem pencatatan pembayaran SPP secara manual menggunakan kartu pembayaran agar pengelolaan data menjadi lebih efisien dan terorganisir.

## Features
- Login pengguna
- Halaman beranda dengan tampilan gedung sekolah
- Pengelolaan data pengguna (petugas dan siswa)
- Pengelolaan data SPP per kelas
- Form transaksi pembayaran SPP
- Laporan dan riwayat pembayaran SPP siswa
